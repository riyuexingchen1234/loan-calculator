# 贷款真相计算器 💰

> 扒开贷款包装术语，用IRR精确计算真实年化利率

## 一句话介绍

中国贷款市场充斥着各种包装术语——"月费率0.3%"、"日息万五"、"手续费率0.6%"……这些说法让你误以为利率很低，但实际年化可能是宣传的 **1.8~2.4倍**。

本技能通过精确计算 **IRR（内部收益率）**，将所有贷款方案还原为统一标准的**真实年化利率**，让你看清每个产品的真实成本。

## 核心能力

| 能力 | 说明 |
|------|------|
| 🔢 **IRR精确计算** | Newton-Raphson迭代法，小数点后两位都准 |
| 📊 **多方案对比** | 一键对比多个贷款方案，按真实成本排序 |
| 🚨 **套路识别** | 自动识别砍头息、虚报利率等常见陷阱 |
| 💬 **自然语言输入** | "借呗借1万分12期每期还880"直接算 |
| 🏦 **全场景覆盖** | 信用卡分期、网贷、房贷、气球贷、随借随还 |

## 为什么你需要它？

**场景1：信用卡分期**
> "月费率0.6%，借1万12期" → 你以为年化7.2%？
> 
> **真相：真实年化 13.84%，是宣传的 1.9倍！**

**场景2：网贷砍头息**
> "借5000到手4000，分12期每月还500"
> 
> **真相：真实年化 121.84%，属于高利贷！**

**场景3：日息包装**
> "日息万五，随借随还" → 看起来很低？
> 
> **真相：年化 18.25%，远超银行信用贷**

## 支持的贷款类型

- ✅ 等额本息（银行信用贷、消费贷）
- ✅ 等额本金（房贷常用）
- ✅ 先息后本（经营贷）
- ✅ 随借随还（借呗、微粒贷、京东金条）
- ✅ 信用卡分期（手续费型）
- ✅ 砍头息（服务费前置扣除）
- ✅ 气球贷（末期大额还款）
- ✅ 反推APR（已知月供反推真实利率）

## 计算公式

**真实年化利率 APR = (1+r)^12 - 1**（复利年化，符合央行〔2021〕3号公告）

| 宣传术语 | 你以为的利率 | 真实年化 | 倍数 |
|---------|-------------|---------|------|
| 月费率0.3% | 3.6% | 6.78% | ×1.9 |
| 月费率0.5% | 6.0% | 11.46% | ×1.9 |
| 日息万五 | 0.05%/天 | 18.25% | — |
| 砍头息2%+合同6% | 6% | 14.74%+ | ×2.4 |
