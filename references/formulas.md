# 计算公式参考

## 1. IRR（内部收益率）

### 定义
IRR 是使净现值（NPV）等于零的折现率：

```
NPV = Σ(CF_i / (1+r)^i) = 0
```

其中：
- CF_i = 第 i 期的现金流（负数=到手金额，正数=还款）
- r = 单期利率
- i = 期数

### Newton-Raphson 求解法

```
r_{n+1} = r_n - NPV(r_n) / NPV'(r_n)

其中:
NPV(r) = Σ(CF_i / (1+r)^i)
NPV'(r) = Σ(-i × CF_i / ((1+r)^(i+1)))

迭代直到 |r_{n+1} - r_n| < 1e-10
```

### 真实年化利率（APR）

```
按月还款: APR = (1+r)^12 - 1  （复利年化，符合监管标准）
按日计息: APR = (1+r)^365 - 1  （复利年化）
```

## 2. 等额本息

### 月供公式
```
M = P × r × (1+r)^n / ((1+r)^n - 1)
```

### 推导
设每月还款额为 M，则：
```
P = M/(1+r) + M/(1+r)^2 + ... + M/(1+r)^n
P = M × [(1+r)^n - 1] / [r × (1+r)^n]
M = P × r × (1+r)^n / [(1+r)^n - 1]
```

### 每期构成
```
利息_i = 剩余本金_{i-1} × r
本金_i = M - 利息_i
剩余本金_i = 剩余本金_{i-1} - 本金_i
```

## 3. 等额本金

### 公式
```
每月本金 = P / n
利息_i = (P - (i-1)×P/n) × r
还款_i = 每月本金 + 利息_i
```

### 特点
```
首月还款 = P/n + P×r
逐月递减 = P/n × r
总利息 = P×r×(n+1)/2
```

## 4. 月费率 → IRR

### 现金流
```
CF_0 = -P
CF_i = P/n + P×f  (i=1,2,...,n)
其中 f = 月费率
```

### IRR 求解
```
-P + Σ[(P/n + P×f)/(1+r)^i] = 0
-P + P×Σ[1/n + f/(1+r)^i] = 0
-1 + Σ[1/n + f/(1+r)^i] = 0
```

### 近似公式
```
APR ≈ 月费率 × 12 × 1.8

精确值 vs 近似值误差 < 5%
```

## 5. 日息 → 年化

```
APR = 日利率 × 365

日息万五 = 0.0005
APR = 0.0005 × 365 = 18.25%

日息万三 = 0.0003
APR = 0.0003 × 365 = 10.95%
```

## 6. 砍头息 → IRR

### 现金流
```
CF_0 = -(P - 服务费) = -(P - P×s) = -P×(1-s)
CF_i = P/n + P×f  (i=1,2,...,n)
其中 s = 服务费比例，f = 月费率
```

### IRR 求解
```
-P×(1-s) + Σ[(P/n + P×f)/(1+r)^i] = 0
```

⚠️ **若用户已知道真实每期还款额M（最常见场景）**，不能套用上面的f重新算M，
应直接用真实M构造现金流求IRR：
```
CF_0 = -P×(1-s)   （到手金额）
CF_i = M           （用户给出的真实月供，i=1,...,n）
```

## 7. 气球贷 → IRR

### 月供反解
贷款本金 = 月供的年金现值 + 尾款的现值：
```
P = M × [1-(1+r)^-n]/r + B×(1+r)^-n

其中：
B = P × 尾款比例
反解：M = (P - B×(1+r)^-n) / {[1-(1+r)^-n]/r}
```

⚠️ **不能**把本金拆成 `摊销本金 = P - B` 后单独对摊销本金做等额本息计算——
那样会导致反推APR远低于真实利率（实测偏差可达40%以上）。
整个贷款期内，计息基数一直是全额本金P，不能提前剔除尾款部分。

### 现金流（用于反推真实APR）
```
CF_0 = -P
CF_i = M           (i=1,...,n-1)
CF_n = M + B       (最后一期=月供+尾款)
```
