# 常见贷款套路术语对照表

## 术语速查

| 宣传术语 | 字面意思 | 真实含义 | 换算公式 |
|---------|---------|---------|---------|
| 月费率0.3% | 每月0.3% | 实际年化约6.58% | APR ≈ 月费率×12×1.8 |
| 月息X厘 | 每月X厘 | 1厘=0.1%，X厘=X×0.1% | 月费率 = X/1000 |
| 日息万X | 每天万X | 年化=万X×0.01%×365 | APR = 日利率×365 |
| 日息万分之X | 同上 | 同上 | 同上 |
| 手续费率X% | 每X% | 按全额收，非按剩余本金 | 同月费率 |
| 先息后本 | 先还利息 | 到期一次性还大额本金 | 表面利率≠真实成本 |
| 随借随还 | 随时可还 | 按日计息，无提前还款惩罚 | 日利率×365 |
| 砍头息 | 服务费前置 | 实际到手<合同金额 | APR = IRR(实际到手→月供) |
| 贴息 | 银行补贴利息 | 可能隐藏其他费用 | 需计算综合成本 |
| 零利率/免息 | 不收利息 | 收手续费/服务费 | 手续费≈真实利率 |

## 详细解析

### 月费率 vs 年化利率

**月费率**是贷款机构最常用的包装手段。

举例：借款10000元，月费率0.3%，12期

- 每月手续费 = 10000 × 0.3% = 30元
- 每月本金 = 10000 / 12 = 833.33元
- 每月还款 = 863.33元
- 总手续费 = 360元
- 宣传年化 = 0.3% × 12 = 3.6%
- **真实年化（IRR） = 6.58%**

**为什么差这么多？**
因为你第一个月欠10000元，交30元手续费。
第十二个月只欠833元，还是交30元手续费。
手续费没有随着本金减少而减少。

### 日息万X

**日息万五**是最常见的网贷利率。

- 日息万五 = 0.05% = 0.0005
- 真实年化 = 0.0005 × 365 = 18.25%

很多人以为"一天才0.05%，很低"，但一年365天就是18.25%。

### 先息后本

**表面利率极低**的典型。

举例：借款100000元，月息0.4%，12期先息后本

- 每月利息 = 100000 × 0.4% = 400元
- 每月还款 = 400元
- 到期还本金 = 100000元
- 总利息 = 4800元
- 宣传年化 = 4.8%
- **真实年化 = 4.8%**（表面数字是真的）

但问题在于到期要一次性还10万。如果还不起，只能续贷：
- 续贷手续费 = 100000 × 0.5% = 500元
- 实际年化 = 4.8% + 续贷成本 ≈ 6%+

### 砍头息

**最恶劣的套路。**

举例：借款50000元，服务费3%，月费率0.5%，12期

- 服务费 = 50000 × 3% = 1500元
- 实际到手 = 48500元
- 每月还款 = 50000/12 + 50000×0.5% = 4666.67 + 250 = 4916.67元
- 总还款 = 59000元
- 实际多付 = 59000 - 48500 = 10500元
- 宣传年化 = 6%
- **真实年化（IRR） ≈ 12.1%+**

### 零利率/免息分期

常见于信用卡购物分期、车贷。

- 不收利息，但收手续费
- 手续费率通常0.5%-0.6%/月
- 真实年化 = 0.5% × 12 × 1.8 ≈ 10.8%

### 综合资金成本

正规银行会披露 APR，包含：
- 利息
- 手续费
- 担保费
- 保险费
- 账户管理费

**所有费用加起来除以本金再年化，才是真实成本。**

## 防骗口诀

1. **月费率 × 12 × 1.8 ≈ 真实年化**（快速估算）
2. **日息万五 = 年化18.25%**（记住这个数字）
3. **先息后本看到期，续贷成本要算清**
4. **砍头息最坑，到手少于借的**
5. **零利率≠零成本，手续费就是利息**
6. **IRR 是唯一标准，其他都是包装**
